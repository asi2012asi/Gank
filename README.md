# Gank

A new Flutter application for gank.io .

## Features

* The latest ganks.
* The history ganks.
* The welfares.
* Submit ganks.
* ...

## What can you get?

* The basis of flutter.
* Custom zoom box for image or any child widget.
* Refresh and load more of the ListView/GridView.
* ...
